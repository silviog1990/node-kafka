# node-kafka
> Producer and Consumer kafka with kafkajs

> kafka kafkajs producer consumer

## Installation

- Require kafka installed (see `https://kafka.js.org/docs/running-kafka-in-development`)

### Setup

> install dependencies

```shell
$ npm install
```

## Usage

### Producer

```shell
$ npm run producer
```

### Consumer

```shell
$ npm run consumer
```
